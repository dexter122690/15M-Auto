# 15M Auto — BodyWorks Operations MVP

A responsive, self-contained operations prototype for an auto body and paint shop.

Open `index.html` in a browser. The app includes the seeded Nilo Flanco invoice:

- Full Washover + Strip — PHP 63,500.00
- PHP 30,000.00 paid
- PHP 33,500.00 accounts receivable

It demonstrates connected invoice, payment, work-order, inventory, cash-basis revenue, reporting, and audit-log flows.
